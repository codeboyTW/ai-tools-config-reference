# AI Tools Config Reference

> A reference guide for Prompt / Skills / MCP configuration paths across Claude CLI, GitHub Copilot, OpenAI Codex CLI, and Google Gemini CLI

---

## Overview

This document maps out the configuration paths for four major AI coding assistants, covering:

- **Prompt / Instruction files** — Global and project-level instruction file locations
- **Skills** — Paths and formats for reusable custom skills (`SKILL.md`)
- **MCP configuration** — Model Context Protocol server config paths and examples
