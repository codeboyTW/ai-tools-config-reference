## 內容說明

本文件整理四大 AI 編程助手的配置路徑，涵蓋：

- **Prompt / 指令檔**：全局與專案級指令檔案位置
- **Skills**：自訂可重用技能（`SKILL.md`）的路徑與格式
- **MCP 配置**：Model Context Protocol 伺服器配置路徑與範例
